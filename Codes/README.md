all the scripts for trainining of seq-seq model, cvae model, segnet model for objective evaluation are present here.

`config` folder has the configurations and paths for experiments <br>
`models` folder contains implementation of all the models used in the work <br>
`data_prep` folder has the code to create custom data loaders for each setup <br>
`common` folder has some utilities such as mapping config to the experiment codes, logging etc
